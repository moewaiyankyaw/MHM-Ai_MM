# M.H.M AI Telegram Bot

**M.H.M AI** သည် မြန်မာနိုင်ငံမှ Grade 11 ကျောင်းသားတစ်ဦးမှ ဖန်တီးထားသော AI-powered Telegram စကားပြောဘော့(ခ်) ဖြစ်ပြီး၊ မြန်မာဘာသာဖြင့် အသုံးပြုနိုင်ပြီး ရာသီဥတုခန့်မှန်းချက်များ၊ ပညာရပ်ဆိုင်ရာအကူအညီများနှင့် စကားပြောအတွေ့အကြုံကို အပြည့်အဝထောက်ပံ့ပေးသည့် အက်ပ်လီကေးရှင်းဖြစ်ပါသည်။

---

## 🚀 အဓိကလုပ်ဆောင်ချက်များ

- 🤖 **AI စကားပြောစနစ်** – သင့်နှင့် စကားပြောမှုများကို မှတ်သားထားပြီး context-aware အဖြေများပေးပို့ပါသည်။
- 🌦️ **ရာသီဥတုခန့်မှန်းချက်** – မြန်မာနိုင်ငံရှိ မည်သည့်တည်နေရာအတွက်မဆို ရာသီဥတုအချက်အလက်များကို ရယူပေးပါသည်။
- 📚 **ပညာရပ်ဆိုင်ရာအကူအညီ** – ဓာတုဗေဒ၊ ရူပဗေဒ၊ ဇီဝဗေဒ၊ အင်္ဂလိပ်စာ စသည့်ဘာသာရပ်များအတွက် အကူအညီပေးပါသည်။
- 💬 **မြန်မာဘာသာဖြင့်အသုံးပြုမှု** – အသုံးပြုသူအားလုံးအတွက် နားလည်လွယ်ကူသော မြန်မာဘာသာဖြင့် တုံ့ပြန်ပေးပါသည်။
- 🔐 **ဒေသထုတ် AI နည်းပညာ** – OpenAI မဟုတ်ဘဲ၊ မြန်မာနိုင်ငံမှ ဖန်တီးထားသော API များကို အသုံးပြုထားပါသည်။

---

## 🛠️ အသုံးပြုသည့်နည်းပညာများ

- **Python** – ပရိုဂရမ်ဘာသာရပ်
- **python-telegram-bot** – Telegram Bot အတွက် API ကိုင်တွယ်ရန်
- **Requests** – HTTP API များနှင့်ချိတ်ဆက်ရန်
- **Asyncio** – စီးဆင်းမှုများကို မပိတ်ဆို့ဘဲ အလုပ်လုပ်ရန်
- **External APIs**:
  - `api.paxsenix.org` – AI စကားပြောအတွက်
  - `api.paxsenix.biz.id` – တည်နေရာကိုဩဒိနိတ်ရှာရန်
  - `weather-forcast.moewaiyankyaw353.workers.dev` – ရာသီဥတုအချက်အလက်များအတွက်

## 📦 အသုံးပြုပုံ

### 1. စတင်ရန်
/start
ကြိုဆိုစာကိုပြပေးပါမည်။

### 2. အကူအညီရယူရန်
/help
အသုံးပြုနိုင်သော commands များကို ဖော်ပြပေးပါမည်။

### 3. စကားပြောမှတ်တမ်းရှင်းလင်းရန်
/clear
AI ကို စကားအသစ်စတင်ရန် ပြင်ဆင်ပေးပါမည်။

### 4. ဖန်တီးသူအကြောင်းကြည့်ရန်
/developer
ဘော့(ခ်) ကို ဖန်တီးသူအကြောင်း ပြပေးပါမည်။

### 5. ရာသီဥတုမေးမြန်းရန်
/weather ရန်ကုန်
ရန်ကုန်မြို့၏ လက်ရှိရာသီဥတုကို ပြပေးပါမည်။

---

## ⚙️ အလုပ်လုပ်ပုံအဆင့်ဆင့်

1. **အသုံးပြုသူမက်ဆေ့ချ်ပို့သည်**  
   → မက်ဆေ့ချ်ကို bot က လက်ခံပါသည်။

2. **စာသားအမျိုးအစားစစ်ဆေးသည်**  
   → Command ဖြစ်ပါက သင့်လျော်သော function ကို ခေါ်ပါသည်။  
   → စာသားဖြစ်ပါက AI ကို အဖြေတောင်းပါသည်။

3. **AI အဖြေပြန်ပေးခြင်း**  
   → စကားပြောမှတ်တမ်းကို ထိန်းသိမ်းပြီး၊ API မှတစ်ဆင့် အဖြေရယူပါသည်။  
   → "စဉ်းစားနေသည်..." မက်ဆေ့ချ်ကို ယာယီပြပေးပါသည်။

4. **ရာသီဥတုစုံစမ်းခြင်း**  
   → `/weather` command တွင် တည်နေရာကို ကိုဩဒိနိတ်ရှာရန် API သို့ပို့ပါသည်။  
   → ရလာသော latitude/longitude ဖြင့် ရာသီဥတု API မှ ဒေတာရယူပါသည်။  
   → ဖော်မတ်ပြီး မြန်မာဘာသာဖြင့် ပြပေးပါသည်။

5. **အဖြေပြန်ပို့ခြင်း**  
   → အဖြေကို မြန်မာဘာသာဖြင့် ပြန်ပို့ပြီး watermark ထည့်ပေးပါသည်။
